About Game
DeadByte: BlackMan Edition is a fast-paced real-time multiplayer browser game with robot characters, voice communication, sound effects, and smooth combat gameplay.
How To Play
1) Choose Mode (Group or Single).
2) For Group: type a Room Code and select your Name.
3) Click Join to start.
4) Move with WASD/Arrow, aim with Mouse, shoot with F.
5) Press Enter to chat (typing disables shooting).
6) Turn on Voice for mic (Group only).
